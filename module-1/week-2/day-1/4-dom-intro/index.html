DOM Intro

The D.O.M is an A.P.I (Application Programming Interface) :
Set of rules, functions and properties, designed to help coding you apps faster.
The DOM allows you to control very precisely any HTML document !!!

The DOM allows you to select tags from the HTML document and use them in your scripts.
Each tag in a DOM script is translated as an Object.
All the object inherit of the Element Interface.

Each Element object has predefined (DOM) properties :
(like style, textContent, classList, className, id, innerHTML).

Some properties are read-only (you cannot modify them).
Most of them are editable !
